# Everlab — Graphic Elements

Graphic elements support and extend the core Everlab brand. They should be
used sparingly and remain minimal to maintain a clean and structured visual
system.

---

## Main Graphic Element — The Marker

The primary graphic element is a **dot enclosed in a square** — derived from
the biomarker / data point concept at the heart of the Everlab brand.

### Variants
- **Filled** — white dot on black square (for use on light backgrounds)
- **Outlined** — black dot in outlined square (for use on light or crème backgrounds)

### Usage
- Used as a label indicator (e.g. "INSIGHT", "QUESTION 1", "EVERLAB'S APPROACH")
- Anchors the start of section labels in marketing layouts
- Used as a data point terminal on lines and connectors
- Always appears at the same weight/size as the accompanying label text
- Never used decoratively at large scale — keep it small and purposeful

---

## Supporting Graphic Elements — Lines & Connectors

Thin **1px lines** are used to connect markers, divide sections, and create
visual flow across layouts. These are a sub-brand extension and should be
used sparingly.

### Rules
- Always 1px stroke weight
- Colour: black on light / white on dark / orange (`bg/brand/orange/100`) for accent moments
- Can be straight (horizontal or vertical) or curved/arc-shaped
- Always terminate at a marker (dot-in-square) at one or both ends
- Never use thick lines or decorative strokes

### Usage contexts
- Horizontal dividers between content sections
- Connecting data points in infographic layouts
- Vertical lines used as spacers or content separators
- Arc/curve lines used in data visualisation and biomarker journey diagrams

---

## Supporting Graphic Elements — Circles & Arcs

Overlapping circles and arc forms are used in more complex graphic moments —
primarily for data visualisation, infographics, and editorial layouts.

### Rules
- Always thin stroke (1px), never filled
- Black on light / white on dark
- Used to suggest cycles, patterns, and health data journeys
- Never use as decorative background elements
- Scale should feel considered — not too large, not too small

## Infographic & Data Visualisation System

Designed primarily for marketing and educational content. The goal is to
present health insights in a clear, approachable, and visually engaging way.

> B2B materials, internal reports, or clinical documentation may follow a
> separate, more structured data visualisation approach.

### Infographic layout principles
- Use the marker element to label data categories and section headers
- Combine photography with graphic overlays for editorial infographics
- Use crème or dark colourways as background — avoid plain white for infographics
- Data labels use Saans Mono (Label Mono) for clinical precision feel
- Support copy uses `fg/neutral/200` (60% opacity) for secondary information
- Cards and panels use subtle borders — `br/neutral/tertiary/100`

### Infographic card style
- Rounded corners, subtle border
- Crème or dark background
- Marker element top-left corner for category label
- Headline + supporting body copy
- Saans Mono for any data or measurement values

---

## Section Label Treatment

Consistent across all marketing layouts — used to introduce sections and
content categories.

Structure: `[marker] LABEL TEXT IN CAPS`

- Marker: dot-in-square at ~12px
- Text: Saans, Label Body 100–200, Medium cap, `fg/neutral/200` (60% opacity)
- Colour: orange marker (`bg/brand/orange/100`) on light backgrounds
- Always uppercase, never sentence case for labels
