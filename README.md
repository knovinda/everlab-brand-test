# Everlab Brand System

A structured brand reference for the Everlab growth team. This repo is the single source of truth for how the Everlab brand looks, sounds, and behaves across marketing, content, and communications.

> **Scope:** This is a brand system for growth — not a product design system. Product has its own component library and token structure. This sits alongside that, not on top of it.

---

## How to use with Claude Design

1. Open Claude Design (claude.ai)
2. Reference this repo URL: `https://github.com/knovinda/everlab-brand-test`
3. Point Claude to the relevant `.md` file for your task — e.g. "use the colours and typography guidelines from this repo to design a landing page section"
4. Claude will read the brand rules and apply them to whatever you're building
